# Homebrew Inrimage

## Installation
```bash
brew tap inrimage/inrimage
brew install inrimage
```
To avoid the building of gvis:
```bash
brew install inrimage --without-gvis
```
To enable the support of hdf5:
```bash
brew install inrimage --with-hdf5
```
